# 🧋 珍珠武士 (Boba Survivors) - 遊戲總論

## 📖 簡介

《珍珠武士》是一款以「珍珠奶茶」為主題的 Roguelike 生存動作遊戲。玩家扮演一名守護萬事屋的珍珠武士，在面對不斷湧入的甜點怪獸（布丁、芋圓、抹茶）中生存，並透過強化珍珠威力來戰勝強大的「十殿閻君」。

## 🎮 控制指南 (手遊版優化)

- **動態移動**：在螢幕左側區域任何位置按下，搖桿中心會隨指尖出現。
- **自動發射**：珍珠武士會自動朝最近的敵人發射珍珠彈。
- **升級選擇**：升級時會彈出材料選單，建議根據戰況平衡攻擊與功能。

## ⚔️ 核心系統

- **等級成長**：每次升級會自動獲得 **HP +2** 與 **ATK +2** 的基礎體質成長。
- **十殿閻君 (Boss System)**：每分鐘的第 50 秒，一隻強大的首領將會橫空出世。擊敗首領可獲得 **黃金經驗珍珠 (10 XP)**。
- **無限混沌模式**：
  - 前 20 階段 (S1-S20) 為關卡制。
  - 20 層後進入**無限大亂鬥**，場景會隨著每一波怪獸重生而隨機變化，挑戰玩家的應變極限。

## 🎨 視覺規格

- **背景對比**：採用「盛夏青綠草皮」，透過高對比度確保黃金珍珠與黑色子彈清晰可見。
- **圖標系統**：全介面使用自定義 .webp 圖標，徹底解決 Web 匯出版的 Emoji 亂碼問題。
- **字體渲染**：掛載 `font.ttf` 確保繁體中文在所有平台都能正確顯示。
