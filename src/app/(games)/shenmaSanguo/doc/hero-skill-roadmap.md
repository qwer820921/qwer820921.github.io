## 2. 技能實作評估

### A. 做得到且好做 (優先級：高)
這些技能主要涉及數值調整或簡單狀態切換。

| 技能名稱 | 類型 | 描述 | 技術實作點 |
| :--- | :--- | :--- | :--- |
| **閃避提升** | Passive | 提升15%閃避率 | 在 `take_damage` 加入 `randf()` 判斷 |
| **百步穿楊** | Passive | 超遠攻擊距離 | 調整 `attack_range` 變數 |
| **堅韌** | Passive | 低血量時獲得減傷 | 判斷 HP 比例觸發防禦倍率 |
| **補給** | Passive | 增加擊殺資源獲取 | 勾掛 `enemy_died` 加金幣邏輯 |
| **守護** | Aura | 提升基地防禦 | 在 `BattleManager` 扣血處加判斷 |
| **減速光環** | Aura | 周圍敵人減速10% | 使用 `Area2D` 偵測並修改敵人 `speed_mult` |
| **火攻** | DOT | 持續燒傷 | 在敵人端加上 `burn_timer` 計時器 |
| **連射** | Trigger | 機率性二次攻擊 | 攻擊後機率重置 `_atk_timer` |
| **奇襲/衝鋒** | Trigger | 首擊必殺或雙倍傷害 | 使用 `first_hit_done` 旗標控制 |
| **戰神** | Scaling | 攻擊力隨殺敵數增加 | 監聽擊殺信號並累加 `atk_bonus` |

### B. 做得到但不好做 (優先級：中)
涉及較複雜的路徑計算或周圍目標掃描。

| 技能名稱 | 類型 | 描述 | 技術實作點 |
| :--- | :--- | :--- | :--- |
| **怪力 (擊退)** | CC | 將敵人往路徑後方推 | 需計算座標並修正敵人的 `_wp_index` |
| **連環計** | AoE | 傳遞傷害給周圍敵人 | 傷害發生時掃描目標圓周內的敵人群 |
| **反擊** | Trigger | 受傷時反彈傷害 | 需要追蹤傷害來源 (Source Tracking) |
| **呼風喚雨** | Active | 大範圍高傷害技能 | 需配合主動技能系統與 VFX 特效 |

### C. 難度極高/需動架構 (優先級：低)
需要修改敵人 AI 或全域傷害分配系統。

| 技能名稱 | 類型 | 描述 | 技術挑戰 |
| :--- | :--- | :--- | :--- |
| **魅惑** | CC | 控制敵人反戈相向 | 需要建立敵對陣營 AI 與目標尋找邏輯 |
| **護衛** | Passive | 替隊友分擔傷害 | 需要「全域傷害分流系統」，架構變動大 |

## 3. 推薦武將組合 (Synergy)

*   **五虎將 (關羽/張飛/趙雲/馬超/黃忠)**: 同時在場時觸發「威壓」光環。
*   **桃園三結義 (劉/關/張)**: 提升防禦與回血速度。
*   **臥龍鳳雛 (諸葛亮/龐統)**: 提升全體主動技能冷卻速度。

### 具體武將技能配對建議：
- **關羽**: 戰神 (殺敵成長) + 威壓 (降敵攻)
- **張飛**: 怪力 (擊退) + 暈眩 (控制)
- **諸葛亮**: 呼風喚雨 (大範圍) + 連環計 (傷害傳遞)
- **趙雲**: 閃避提升 + 奇襲 (首擊必殺)
- **呂布**: 衝鋒 (首擊翻倍) + 戰神 (殺敵成長)
- **黃忠**: 百步穿楊 (超遠程) + 連射 (機率雙發)
- **貂蟬**: 魅惑 (控制敵軍) — *最具挑戰性的特色技能*
- **周瑜**: 火攻 (持續燒傷) + 指揮 (友軍攻速)

## 4. 下一步行動計畫
1.  **實作基礎光環組件**: 在 `Hero.gd` 建立通用的 `Area2D` 偵測器。
2.  **建立狀態系統 (Buff/Debuff System)**: 統籌管理減速、燒傷、暈眩等狀態，避免互相覆蓋衝突。
3.  **開發主動技能通訊協定**: 確立 Web 端按鈕點擊後如何觸發 Godot 內部的技能動畫與效果。
