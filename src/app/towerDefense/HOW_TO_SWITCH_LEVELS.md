# 🎮 如何切換關卡

## 方法 1：從開始畫面選擇關卡

1. **打開遊戲**

   - 進入塔防遊戲頁面

2. **選擇關卡**

   - 在開始畫面會看到「選擇關卡」下拉選單
   - 點擊選單可以看到所有可用關卡
   - 每個關卡會顯示：
     - 關卡編號
     - 關卡名稱
     - 難度等級

3. **查看關卡資訊**

   - 選擇關卡後，下方會顯示：
     - 關卡描述
     - 初始金幣
     - 初始生命值
     - 波次數量

4. **開始遊戲**
   - 點擊「開始遊戲」按鈕
   - 遊戲會載入你選擇的關卡

## 方法 2：通關後前往下一關

1. **完成當前關卡**

   - 擊敗所有波次的敵人

2. **勝利畫面**

   - 會顯示當前關卡名稱和分數
   - 如果有下一關，會出現「下一關 →」按鈕

3. **前往下一關**

   - 點擊「下一關 →」按鈕
   - 遊戲會自動載入下一個關卡並開始

4. **全部通關**
   - 如果已經是最後一關
   - 會顯示「🎊 恭喜通關所有關卡！」

## 方法 3：使用程式碼切換（開發用）

```typescript
import { useGameStore } from "./store/gameStore";

// 在組件中
const { loadLevel } = useGameStore();

// 載入特定關卡
loadLevel(3); // 載入關卡 3
```

## 遊戲中的關卡資訊

### HUD 顯示

遊戲進行時，頂部 HUD 會顯示：

- 🎯 **當前關卡名稱**
- 💰 金幣
- ❤️ 生命值
- 🌊 波次進度
- 🏆 分數

### 關卡特性

每個關卡都有獨特的設定：

- **初始資源**：不同的起始金幣和生命值
- **波次配置**：不同的敵人組合和數量
- **難度等級**：easy、medium、hard、extreme

## 重新開始當前關卡

在遊戲進行中或結束後：

1. 點擊 HUD 上的 🔄 按鈕
2. 或在遊戲結束畫面點擊「再玩一次」
3. 會重新載入當前關卡

## 提示

- 💡 **選擇合適的關卡**：根據你的技能選擇難度
- 💡 **查看關卡資訊**：開始前先了解初始資源
- 💡 **循序漸進**：建議從簡單關卡開始
- 💡 **挑戰自己**：通關後嘗試更高難度

## 開發者：新增關卡

如果你想新增更多關卡：

1. 編輯 `data/levels.json`
2. 添加新的關卡配置
3. 重新啟動開發服務器
4. 新關卡會自動出現在選單中

詳細說明請參考：`data/README.md`
